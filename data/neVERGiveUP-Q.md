# NON-CRITICAL ISSUES
## Unlocked solidity version
The Solidity version that is used in all files that are in scope is `^0.8.16`.
This means that the files can be compiled with any compiler version `>0.8.15` and `<0.9.0`.
A fixed compiler version should be used instead to make sure the contracts are not compiled with an unwanted compiler version
that might behave in an unintended way.

## Lines are too long
There are 10 instances of lines that have a length of over 164 characters.
164 characters should be the maximum line length since it is the maximum amount of characters that GitHub will display on a screen without scrolling.
